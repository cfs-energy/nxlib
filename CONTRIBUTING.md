# Contributing

Nxlib is currently a public archive of an internal tool developed by and for Commonwealth Fusion Systems (CFS). The development of nxlib currently happens internally, and this repository is updated on point releases.

Please feel free to open an issue to report bugs or request new functionality, or start a discussion to ask a general question. New feature requests may be addressed providing that they align with CFS internal needs.
